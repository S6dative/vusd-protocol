Prerequisites

  • Tor — brew install tor or download from torproject.org
  • LND — download from github.com/lightningnetwork/lnd/releases
  • Bitcoin Core (optional) — bitcoincore.org

Setup steps

  1. Run thunder setup to generate torrc and lnd.conf
  2. Copy torrc to ~/.tor/torrc and restart Tor: brew services restart tor
  3. Start LND with the generated config: lnd --configfile=~/.lnd/lnd.conf
  4. Open private channels with relay peers (see documentation)
  5. Run thunder start

Operator security checklist

  • ☐ Tor must be running before starting Thunder Node
  • ☐ LND configured with tor.active=true and nolisten=true
  • ☐ Fund LND wallet with non-KYC Bitcoin only
  • ☐ Store operator seed phrase offline (paper, air-gapped device)
  • ☐ Pay for hosting with Monero — never credit card
