⚡ Thunder Node

Privacy-maximized Lightning relay for VULTD

This installer will place the thunder binary in /usr/local/bin/ so it is available from any Terminal window.

Thunder Node wraps your VUSD transfers in ring signatures, stealth addresses, Tor onion routing, and bulletproof amount commitments — then relays them through a multi-hop private Lightning network. "You can't see thunder."

After installation, open Terminal and run:

thunder setup      # generate Tor + LND configuration
thunder threats    # review all 25 threat mitigations
thunder fees 1000  # preview fee breakdown
thunder start      # start the relay (requires Tor + LND)

Requires macOS 11.0 (Big Sur) or later. Universal binary: runs natively on Intel and Apple Silicon (M1/M2/M3/M4).
